### Task 1.2 — Create `bravos/broker/connection.py` — IBApp class

**File:** `bravos/broker/connection.py`

Write the full IBApp class. Use the skeleton from RESEARCH.md exactly. Key implementation rules:

1. Class: `IBApp(EWrapper, EClient)` — inheritance order matters; EWrapper first so its `__init__` is not required (it has none); EClient `__init__` called explicitly as `EClient.__init__(self, self)`.

2. Constructor signature: `__init__(self, host: str, port: int, client_id: int)` — stores `_host`, `_port`, `_client_id`. Initializes all state. Does NOT connect.

3. State to initialize:
   - `self.next_order_id: int | None = None`
   - `self._connected = threading.Event()`
   - `self._last_heartbeat_at: float = 0.0`
   - `self._reconnecting = False`
   - `self._recon_lock = threading.Lock()`
   - `self._positions: list[dict] = []`
   - `self._open_orders: list[dict] = []`
   - `self._account_summary: dict = {}`
   - `self._positions_done = threading.Event()`
   - `self._orders_done = threading.Event()`
   - `self._summary_done = threading.Event()`
   - `self._stop_event = threading.Event()`
   - `self._api_thread: threading.Thread | None = None`
   - `self._heartbeat_thread: threading.Thread | None = None`

4. `connect_and_run(self, timeout: float = 30) -> bool`: clears `_connected`, calls `self.connect(self._host, self._port, self._client_id)`, starts `ibkr-api` daemon thread running `self.run`, returns `self._connected.wait(timeout)`.

5. `is_connected(self) -> bool`: returns `self._connected.is_set()`.

6. `nextValidId(self, orderId: int)`: sets `self.next_order_id = orderId`, sets `self._last_heartbeat_at = time.monotonic()`, calls `self._connected.set()`, logs INFO with orderId and port.

7. `currentTime(self, time_val: int)`: sets `self._last_heartbeat_at = time.monotonic()` — no logging (called every 60s; noisy).

8. `error(self, reqId: int, errorCode: int, errorString: str, advancedOrderRejectJson: str = "")`: implement error code routing per RESEARCH.md Pattern 4. Use module-level sets `_IGNORE_CODES`, `_CRITICAL_NO_RETRY_CODES`, `_IMMEDIATE_RECONNECT_CODES`.

9. `stop(self)`: sets `self._stop_event`, clears `self._connected`, calls `self.disconnect()` inside try/except, logs INFO.

10. Module-level constants at top of file:
    ```python
    _RETRY_DELAYS = [5, 10, 20, 40, 80]
    _IMMEDIATE_RECONNECT_CODES = {504, 1100}
    _IGNORE_CODES = {2104, 2106, 2119, 2158, 2110}
    _CRITICAL_NO_RETRY_CODES = {501, 507}
    REQ_ID_ACCOUNT_SUMMARY = 9001
    HEARTBEAT_INTERVAL = 60
    HEARTBEAT_TIMEOUT = 10
    ```

11. Module-level singleton at bottom of file:
    ```python
    # Module-level singleton — set by run_ingestion.py at startup, not at import time.
    # Phase 4 callers: from bravos.broker.connection import ibapp
    ibapp: "IBApp | None" = None
    ```

12. Do NOT implement `_heartbeat_loop`, `_trigger_reconnect`, `_reconnect_loop`, `start_heartbeat_monitor`, `start_background_reconnect`, `run_startup_reconciliation`, or any reconciliation helpers in this plan — those are Plans 2 and 3. Leave stubs that raise `NotImplementedError` if needed, or simply don't define them yet.
