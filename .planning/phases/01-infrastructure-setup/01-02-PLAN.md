   ```bash

   sudo apt-get update
   sudo apt-get install -y python3.11 python3.11-venv python3.11-dev
   ```

2. Create venv (per D-13):
   ```bash
   python3.11 -m venv /home/$USER/venv
   source /home/$USER/venv/bin/activate
   pip install --upgrade pip
   ```

3. Copy requirements.txt to VM and install (per D-14):
   ```bash
   pip install -r requirements.txt
   ```

4. Install ibapi from official IB zip (per D-15). Use the TWS API version that matches the IB Gateway version discovered in Plan 01 Task 2 (from opt-trade-vm4 investigation). The exact steps:
   ```bash
   # Download TWS API zip — version number from 01-01-SUMMARY.md
   # Example: twsapi_macunix.1030.01.zip
   wget "https://interactivebrokers.github.io/downloads/twsapi_macunix.XXXX.XX.zip" -O /tmp/twsapi.zip
   unzip /tmp/twsapi.zip -d /tmp/twsapi
   cd /tmp/twsapi/source/pythonclient
   pip install .
   ```
   Replace XXXX.XX with the actual version from the opt-trade-vm4 investigation.
