**Part A: Create config files in repo (infra/ directory)**

1. Create `infra/ibc-config.ini` — IBC config TEMPLATE (no real secrets, per D-16/D-17):
   ```ini
   # IBC Configuration for Bravos Trading System
   # Credentials injected at startup from GCP Secret Manager
   IbLoginId=PLACEHOLDER
   IbPassword=PLACEHOLDER
   TradingMode=paper
   SecondFactorDevice=
   ReloginAfterSecondFactorAuthenticationTimeout=yes
   SecondFactorAuthenticationTimeout=180
   ExitAfterSecondFactorAuthenticationTimeout=no
   AcceptNonBrokerageAccountWarning=yes
   OverrideTwsApiPort=
   ExistingSessionDetectedAction=primaryoverride
   AcceptIncomingConnectionAction=accept
   AllowBlindTrading=no
   ```

2. Create `infra/start-gateway.sh` — startup script that injects secrets from GCP Secret Manager (per D-06, D-16):
   ```bash
   #!/bin/bash
   set -euo pipefail
   export DISPLAY=:1

   # Pull credentials from GCP Secret Manager and inject into IBC config
   IBKR_USER=$(gcloud secrets versions access latest --secret="bravos-ibkr-username")
   IBKR_PASS=$(gcloud secrets versions access latest --secret="bravos-ibkr-password")

   CONFIG=~/ibc/config.ini
   sed -i "s/^IbLoginId=.*/IbLoginId=${IBKR_USER}/" "$CONFIG"
   sed -i "s/^IbPassword=.*/IbPassword=${IBKR_PASS}/" "$CONFIG"

   # Start IB Gateway via IBC
   # TWS_MAJOR_VNUM must match the installed Gateway version (from 01-01-SUMMARY.md)
   TWS_MAJOR_VNUM="REPLACE_WITH_VERSION"  # e.g., "1030" — set from opt-trade-vm4 investigation
   /opt/ibc/scripts/gatewaystart.sh "$CONFIG" ~/Jts/ibgateway/${TWS_MAJOR_VNUM} "${TWS_MAJOR_VNUM}" paper
   ```

3. Create `infra/xvfb.service`:
   ```ini
   [Unit]
   Description=X Virtual Framebuffer
   After=network.target

   [Service]
   Type=simple
   User=REPLACE_WITH_VM_USER
   ExecStart=/usr/bin/Xvfb :1 -screen 0 1024x768x24 -nolisten tcp
   Restart=always
   RestartSec=5

   [Install]
   WantedBy=multi-user.target
   ```

4. Create `infra/ibgateway.service`:
   ```ini
   [Unit]
   Description=IB Gateway (managed by IBC)
   After=network.target xvfb.service
   Requires=xvfb.service

   [Service]
   Type=forking
   User=REPLACE_WITH_VM_USER
   Environment=DISPLAY=:1
   ExecStart=/home/REPLACE_WITH_VM_USER/start-gateway.sh
   Restart=no

   [Install]
   WantedBy=multi-user.target
   ```
   Note: `Restart=no` per RESEARCH.md open question 4 — operator manually starts Gateway to handle 2FA.

**Part B: Install on VM via SSH**

SSH into bravos_vm1 and run:

1. Install Java and Xvfb:
   ```bash
   sudo apt-get update
   sudo apt-get install -y openjdk-11-jre xvfb x11vnc
   ```

2. Download and install IB Gateway (stable):
   ```bash
   wget https://download2.interactivebrokers.com/installers/ibgateway/stable-standalone/ibgateway-stable-standalone-linux-x64.sh -O /tmp/ibgateway-installer.sh
   chmod u+x /tmp/ibgateway-installer.sh
   sudo Xvfb :1 -screen 0 1024x768x24 &
   export DISPLAY=:1
   /tmp/ibgateway-installer.sh -c
   # Accept defaults, install to ~/Jts/ibgateway/
   ```
   If `-c` (console mode) is not supported, use `DISPLAY=:1 /tmp/ibgateway-installer.sh` and accept defaults.

3. Download and install IBC — use the version from opt-trade-vm4 (from 01-01-SUMMARY.md):
   ```bash
   IBC_VERSION="REPLACE_WITH_VERSION"  # From opt-trade-vm4 investigation
   wget -q "https://github.com/IbcAlpha/IBC/releases/download/${IBC_VERSION}/IBCLinux-${IBC_VERSION}.zip" -O /tmp/IBCLinux.zip
   sudo mkdir -p /opt/ibc
   sudo unzip -o /tmp/IBCLinux.zip -d /opt/ibc
   sudo chmod o+x /opt/ibc/*.sh /opt/ibc/scripts/*.sh
   ```

4. Set up IBC config directory:
   ```bash
   mkdir -p ~/ibc
   cp infra/ibc-config.ini ~/ibc/config.ini
   chmod 600 ~/ibc/config.ini
   ```

5. Copy start-gateway.sh to home directory:
   ```bash
   cp infra/start-gateway.sh ~/start-gateway.sh
   chmod 700 ~/start-gateway.sh
   # Update TWS_MAJOR_VNUM with actual Gateway version
   ```

6. Install and enable systemd services:
   ```bash
   sudo cp infra/xvfb.service /etc/systemd/system/xvfb.service
   sudo cp infra/ibgateway.service /etc/systemd/system/ibgateway.service
   # Update User= fields with actual VM username
   sudo systemctl daemon-reload
   sudo systemctl enable xvfb.service
   sudo systemctl start xvfb.service
   ```

7. Verify Xvfb is running:
   ```bash
   systemctl status xvfb.service
   ```
